📡 MoneyRadar API

歡迎使用 MoneyRadar API。本 API 提供全球股票報價、AI 解讀、技術指標、新聞情緒、投組分析等服務。

⚠️ 使用條款：本 API 僅供個人非商業用途。商業使用請聯絡 ceo-nini@agentmail.to。本 API 提供之資料為「公開資訊整理」，不構成投資建議。

🌐 Base URL

https://moneyradar-ai-proxy.thinkbigtw.workers.dev

📝 範例：取得單檔報價

cURL

curl 'https://moneyradar-ai-proxy.thinkbigtw.workers.dev/quote?symbol=NVDA'

JavaScript

const res = await fetch('https://moneyradar-ai-proxy.thinkbigtw.workers.dev/quote?symbol=NVDA');
const data = await res.json();
console.log(data.price, data.changePercent);

Python

import requests
r = requests.get('https://moneyradar-ai-proxy.thinkbigtw.workers.dev/quote', params={'symbol': 'NVDA'})
print(r.json())
